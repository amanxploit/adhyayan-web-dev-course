## 📖 Course Description

This course is designed for beginners who want to become **Full Stack Web Developers** in just **1 month**.

Students will learn:

* HTML (Structure of Web Pages)
* CSS (Styling & Layout)
* JavaScript (Logic & Interactivity)
* React.js (Frontend Framework)
* Node.js (Backend Development)
